## üíæ Context Preservation Protocol (NEW - Sprint 6A)

### üö® Critical Learning: "Don't Exit Without Saving"

**Problem discovered:** Exiting Claude session = losing context, like closing Word without saving.

**Solution:** Check this list BEFORE every session exit:

```
‚òê 1. KEY DECISIONS DOCUMENTED?
   ‚Üí Write to .ai/*.md files (architecture decisions, sprint plans)

‚òê 2. SPRINT STATUS UPDATED?
   ‚Üí Update .ai/BRAIN_PROJECT_SUMMARY.md with current phase

‚òê 3. USER HANDOFF CLEAR?
   ‚Üí What should User do next? (Testing? Feedback?)
   ‚Üí Document in clear terms (User Terminal vs Brain Terminal)

‚òê 4. NEW PATTERNS CAPTURED?
   ‚Üí Add to ~/.claude-workers/brain/patterns/ if discovered

‚òê 5. XP/PROFILE UPDATED?
   ‚Üí Update profile.md if major learning occurred

‚òê 6. NEXT ACTION DOCUMENTED?
   ‚Üí What should next Brain session do?
   ‚Üí Update PROJECT_SUMMARY or create SESSION_HANDOFF.md

‚òê 7. META-INSIGHTS PRESERVED?
   ‚Üí System architecture changes ‚Üí .ai/META_SYSTEM_ARCHITECTURE.md
```
